This is a simple tetris game that can run in cmd.
这是一个简单的俄罗斯方块游戏，可在cmd运行。

Required third-party Python library: keyboard, for implementing keyboard control.
需要的第三方库：keyboard，用于键盘操控。

Edit setup.py to change the settings.
编辑setup.py可修改游戏设置。

Press ↓ for instant fall.
→ for right. ← for left.
p for pause. s for exit.
r for clockwise rotation. l for anticlockwise rotation.
按↓使方块立即下落。
按p暂停游戏，按s退出游戏。
→右移，←左移，r顺时针旋转，l逆时针旋转。
